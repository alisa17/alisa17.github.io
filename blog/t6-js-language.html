()

We use parenthesis to

  1. enclose the parameters given to the function
  2. follow the if/for/while statements to list the conditions for the statement to be executed
  3. to do normal maths or logical prioritisation of operations

[]
We use brackets to
  1. hold an array
  2. when following array name, it contains numerical pointer to a particular element of the array
  3. when follwoing object name, it contains the name of the object property that is being pointed to

{}

Braces are used to

  1. contain code to be executed, such as function code or loop code.
  2. define an object

'' and ""

These are used to contain a string and behave very similar to each other.
