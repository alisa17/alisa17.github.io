How does JavaScript compare to HTML and CSS?

HTML exists to define structure of the contents on the web page, CSS defines its style, while javascript defines the function of various elements on the webpage, how they dynamically intereact with each other and how they respond to user's actions.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow referes to simple actions conducted in sequential manner. An everyday example of a control flow would be an act of waking up:

  1. open your eyes
  2. sit up
  3. get out of bed
  4. brush your teeth
  5. make coffee
  6. drink coffee
  7. study javascript
  8. you're now awake
Loops have to do with the act of going through a collection of things in sequential manner in order to complete a task. An everyday example of a loop would be shopping at a grocery store, looking for flour:
  1. Am I in the baking goods section?
  2. Walk around until I am in the baking goods section
  3. Am I near flour?
  4. Peruse the baking goods isle until I can see flour
  5. Choose flour that I like
  6. The loop is over

Explain the difference between accessing data from arrays and objects.

Accessing data from arrays and objects is very similar in the sense that whenever you want to access data from either an array or an object you have to specify exactly what data you want to access. The difference between the two is that when you specify what data from an array you want to access you specify the numeric order of the elements. When you specify what data you want to access from an object you specify not the numeric order but a qualitative property of the data. For example

  • Give me the first flour on the shelf
  • As opposed to: give me the property 'price' of this flour

Explain what functions are and why they are useful.

Funtion in javascript is a piece of code that accomplishes a single task. They are useful for their reusability - once you have a function that accomplishes a particular task you can always use it in your other code.
