Analogies and Definitions

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML is like furniture inside a room. While CSS is like the designer who comes in and says we put this rag here on this angle, we put the table here and also let's paint this wall purple.
DOM could be described as a well organised wardrobe. There are compartments for various items, inside those compartments there are othe compartments. For example there is a shirt section. Inside the shirt section there are t-shirts, dress shirts and blouses.

What is meant by boxifying design?

It means to create a layout of the webpage while treating every element as a box

What is the box model?

The box model refers to treating an element on a webpage as a box with certain characteristics: margin, border, padding and the element itself
